This data pipeline updates the details of customers or employees in SharePoint when it is used as a database. It uses the WordPress and Squarespace built-in forms or the third-party Formidable Forms to feed responses smartly via Zapier's conditional logic into a Google Sheet. The latter then automatically synchronises with Google Data Studio (Looker) where the customer's data gets updated which then transfers the udated information into SharePoint. Apps: WordPress, Squarespace, Formidable Forms, Zapier, Google Sheets, Google Data Studio (Looker), SharePoint

This web data pipeline connects websites with the Salesforce database making it easy for the sales team to work with website users. It uses the WordPress built-in forms or the third-party Formidable Forms to create popups on the website and uploades user data directly into the Salesforce database via Zapier - creating different categories of potential customers depending on their responses. Apps: WordPress, Formidable Forms, Popup Maker, Zapier, Salesforce

 

This web data pipeline can work independently or in connection with the previous one. It allows the sales team run email marketing campaigns via MailChimp or Constant Contact and update customer data in their Salesforce database authomatically via SyncApps depending on user's actions. Apps: Salesforce, SyncApps, MailChimp, Constant Contact


This web data pipeline connects the forms on Squarespace or WordPress with email marketing tools. It uses Zapier app to analyse the users' responses in the forms and allocates the users into different marketing campaigns in MailChimp or Constant Contact depending on their answers.

In this example, the form on a Squarespace website allows users to opt-in for 4 different email subscriptions. All the possible combinations of choices are allowed.

The data will be fed into email marketing databases and the company will be able to run customised email marketing camapigns depending on particular responses of the website users. Apps: WordPress, Squarespace, Zapier, MailChimp, Constant Contact
